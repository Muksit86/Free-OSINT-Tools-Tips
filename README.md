# 11. Tips, Tricks & Methods

### **General OSINT Tips**

* Always **reverse image search** before trusting a picture.
* Check **first appearance** of an image using TinEye.
* Use **incognito mode** to avoid personalized results.
* Always **cross-check with 2 sources**.
* Translate foreign pages using built-in browser translation.

### **Google Search Tricks**

* Combine operators: `"soviet housing" filetype:pdf site:.edu`
* Use year filters (Tools ‚Üí Time ‚Üí Custom Range)

### **Image OSINT Tricks**

* Crop only a part of the image before reverse searching.
* Upload screenshots instead of original to bypass detection.

### **Video OSINT Methods**

* Extract frames ‚Üí reverse search each frame.
* Check shadows, sun position ‚Üí estimate time of day.

### **Satellite OSINT Techniques**

* Compare imagery from different years.
* Look for: road shadows, car density, construction patterns.
* Use polygon ruler in Google Earth Pro.

### **Documentation Tips for YouTube**

* Create folders for: **images / sources / pdf / maps**
* Maintain a **timeline** doc
* Save **all citations** for video description
